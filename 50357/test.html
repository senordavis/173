Week One:
Basic Text

HTML or Hyper Text Markup Language, is the structure of all pages on the interent. If you were to take a look at the source code of this page you would be looking at HTML. Even though D2L sites are run from a database, all the pages are served dynamically and converted to HTML for your browser to display.

HTML is a language that uses tags. These tags are what give your browser the ability to interpret the content on the page. So when we write our HTML we want to make sure that we write semantically. In essence we want to use tags around our content so that make sense when it is being read.

In this module we will go over how to semantically markup a document using HTML. This will allow you to complete the basic text assignment, which is due

Week One:
Files

One of the other extremely important things to keep in mind when setting up you site, and also for the rest of the semester, is how you name you files. A common pitfall in many people make is failing to understand how important well structured file organization is to your website. In the case of a real world site, it can lead to broken links, lost files, and an unusable site. In the case of our course, the inability to comprehend file management can result in poor grades or even failing the course.

Please pay special attention to the file management demo video where I will show you how to use the files panel inside of Dreamweaver, and a couple of methods for naming your files.
