Site Portfolio

Now that we have learned many of the basic techniques of displaying content, we will start to apply those techniques in many different ways.

We will start out by creating a page that will be the homepage for you webspace.

This page will essentially be a portfolio, or main navigation to all the assignments for this class.

Objectives

  • Create an attractive, well-designed introduction and index to your course work.
  • Include a link to each assignment.
  • Use Dreamweaver roll over images or CSS hover effects appropriately on the page.
  • Follow good practices in HTML and CSS.
  • Upload the page as the home page of your webspace.

Assignment Specifications

Create a portfolio homepage for your web site using Dreamweaver roll over images or CSS hover effects. Each menu item should take the user to one of your other assignments, and every assignment should have a link.

Links for future assignments might bring up a "coming soon" announcement.

You will need to create and optimize your own graphics for this assignment. Beyond the requirement to use Dreamweaver roll over images or CSS hover effects, and link to your course assignments, the design of your page is entirely up to you, so this is a chance to flex your creativity. Visual composition counts, and so do technical aspects like correct markup and image optimization.

File management, file management, file management!

Your site should be organized as described in the File Management Demo Video. Now is the time to clean it up if it is not. While you are cleaning up your files, be sure each of your pages has a title.

File Naming

Save your page with the appropriate name(s), following the file naming convention
(teal items are folders):


 index.html
   assignment-7
   ↳  css
       ↳  style.css
   ↳  images
       ↳  your-own-image.jpg

Note: Your index.html file for this assignment is not located in the assignment folder but rather your root directory outside of all the assignment folders. All the other assignment files should be in the assignment-7 folder.

Once completed, upload your files to your web space on theartprogram.org and to the
Portfolio Dropbox.

Please make sure to upload your files to both your theartprogram.org webspace and the assignment dropbox. This will ensure that you have completed the assignment correctly, and that everything will be graded correctly.

The easiest way to upload your assignment to D2L is to ZIP your assignment folder and upload the ZIP file with all the contents of the folder.
