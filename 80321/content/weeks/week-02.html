Week Two:
Basic Text

Hyper Text Markup Language (HTML), is the structure of all pages on the interent. If you were to take a look at the source code of this page you would be looking at HTML.

HTML is a language that uses tags that give your browser the ability to interpret the content on the page. So when we write our HTML we want to make sure that we write semantically. In other words, we want to wrap tags around our content so that it makes sense when it is read.

In this module we will go over how to semantically markup a document using HTML. This will allow you to complete the basic text assignment, which is due
Saturday, January 21, 2017 at 11:59pm EST.
