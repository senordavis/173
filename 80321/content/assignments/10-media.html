Adding Media

The web has evolved well beyond a place just to share research documents. Now it is a place were all kids of media is published, shared, and viewed. Understanding how this media can be leveraged to create unique designs is vitally important for designers.

Sites like Youtube, Vimeo, and Soundcloud, have given nearly anyone the ability to create and share media very easily. So easily in fact that adding media to sites now is almost expected.

Objectives

  • Media is properly installed in the page.
  • Page design is appropriate for the chosen media file.

Assignment Specifications

Design and build an appropriately designed home for a movie or some other media file. Make your page and your selected media look like they where made for each other.

You may use your own media files, provided that you are legally able to use the files. If you need a help finding where to look for media files, please let me know.

File Naming

Save your page with the appropriate name(s), following the file naming convention
(teal items are folders):


   assignment-10
   ↳  index.html
   ↳  css
       ↳  screen.css
   ↳  images
       ↳  your-own-image.jpg
       ↳  your-own-movie.mp4
       ↳  your-own-audio.mp3

Please make sure to upload your files just to the remote server.
This will ensure that you have completed the assignment correctly, and that everything will be graded correctly.
