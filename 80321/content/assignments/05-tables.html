Working with Tables

When using creating content for you pages, there will be many times when you have different information that needs to be displayed in a different format. One of those types of content is tabular data.

When you have tabular data you should use an HTML table. Tables can be used to display data in rows and columns. In addition, you can also create HTML tables to organize information on your web page.

The process of creating an HTML table is similar to the process that you used to create your web page and any elements that you may have already included in your page, such as images or links.

Note: Prior to the creation of CSS, HTML table elements were often used as a method for page layout. This usage has been discouraged since HTML 4, and the table element should not be used for layout purposes.

Objectives

  • Demonstrate appropriate use of tables.
  • Use caption tag to identify table.
  • Use header tags to distinguish column and row headings from cell text.
  • Do any styling with CSS.
