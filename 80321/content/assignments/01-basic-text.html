Objectives

Create an about me page using semantic markup and upload it to your web space.

Assignment Specifications

Your about page may or may not be truthful and it need not have graphics or other bells and whistles. But it must us good markup.

Your page will use the default styling that the browser will give it. However if you are already knowledgeable enough to use more sophisticated CSS or to add elements to your about page, you certainly may do so—but the assignment is to produce a page that uses correctly marked-up text.

File Naming

Save your page with the appropriate name(s), following the file naming convention
(teal items are folders):


   assignment-1
   ↳  index.html

Please make sure to upload your files just to the remote server.
This will ensure that you have completed the assignment correctly, and that everything will be graded correctly.

If you are having problems please contact me.
